## Telegram credentials

Секреты Telegram вынесены в локальный файл `config/telegram_credentials_local.py`.

1. Скопируйте шаблон:
	- `config/telegram_credentials_local.example.py` -> `config/telegram_credentials_local.py`
2. Заполните значения `TELEGRAM_API_ID`, `TELEGRAM_API_HASH`, `TELEGRAM_SESSION`.

Файл `config/telegram_credentials_local.py` добавлен в `.gitignore` и не попадет в репозиторий.
